# Wiki content (for GitHub wiki)

Copy these files into the [repo wiki](https://github.com/lorddummy/deadlock-anti-cheat/wiki):

1. On GitHub: **deadlock-anti-cheat** → **Wiki** → **New Page** (or edit **Home**).
2. Create a page named **Roadmap** and paste the contents of `Roadmap.md`.
3. Optionally set **Roadmap** as the wiki sidebar or home.

The wiki is the canonical place for the roadmap; the same content lives in `UrnItAnticheat-main/ROADMAP.md` in the repo.
